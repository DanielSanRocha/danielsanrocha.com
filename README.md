# danielsanrocha.me

> My personal Home Page =)

## Build Setup

``` bash
# install dependencies
npm install

# build
npm run build

# serve on port 5000
node server.js
```

## Deploying to Production

You can deploy this server to heroku simpling by pushing this rep to a heroku repository.


## Developing

``` bash
# serve with hot-reload
npm run dev
```
